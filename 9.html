HTML Text Formatting Tags

  1. <b> - Bold
    • Use to make text bold.
    • Best practice is to use it for important keywords or phrases, not for styling purposes.
  2. <em> - Emphasis
    • Indicates text that should be emphasized, often displayed in italics.
    • Best practice is to use it for emphasizing the meaning of the text.
  3. <u> - Underline
    • Historically used to underline text, but not recommended for general use due to potential confusion with hyperlinks.
    • Best practice is to use CSS for text decoration and styling.
  4. <s> - Strikethrough
    • Represents text that is no longer accurate or relevant.
    • Best practice is to use it to indicate deleted or deprecated content.
  5. <mark> - Marked Text
    • Used to highlight or mark parts of text.
    • Best practice is to use it to indicate portions of text relevant to the user's current task or search.
  6. <sub> - Subscript
    • Used to display text as subscript, typically for mathematical or chemical formulas.
    • Best practice is to use it for formatting purposes where subscript is necessary.
  7. <sup> - Superscript
    • Used to display text as superscript, often for footnotes or exponents.
    • Best practice is to use it for formatting purposes where superscript is necessary.
