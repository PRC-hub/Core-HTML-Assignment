HTML TAGS

  1. html:- Represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element.
  2. base:- Specifies the base URL to use for all relative URLs in a document. There can be only one such element in a document.
  3. head:- Contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
  4. link:- Specifies relationships between the current document and an external resource. This element is most commonly used to link to CSS but is also used to establish site icons (both "favicon" style icons and icons for the home screen and apps on mobile devices) among other things.
  5. title:- Defines the document's title that is shown in a browser's title bar or a page's tab. It only contains text; tags within the element are ignored.
  6. body:- represents the content of an HTML document. There can be only one such element in a document.
  7. address:- Indicates that the enclosed HTML provides contact information for a person or people, or for an organization.
  8. footer:- Represents a footer for its nearest ancestor sectioning content or sectioning root element. A footer tag typically contains information about the author of the section, copyright data, or links to related documents.
  9. header:- Represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements.
  10. nav:- Represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes.
